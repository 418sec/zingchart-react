# zingchart-react

> ZingChart React Component


## Install

```bash
npm install --save zingchart-react
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'zingchart-react'

class Example extends Component {
  render () {
    return (
      <MyComponent />
    )
  }
}
```
